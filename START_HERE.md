## 🚀 3-Step Quick Start (Deploy Now!)

### STEP 1: Update Configuration (2 minutes)
```javascript
// Edit config.js
const QUIZ_CONFIG = {
    gumroadUsername: 'your_username',  // Your Gumroad username
    gumroadProductId: 'your_product_id', // Your product ID
    price: '€1.99',  // Your price
};
```

### STEP 2: Update Domain References (5 minutes)
Replace `https://heartbreakquiz.com` with your domain in:
- `index.html`
- `quiz.html` 
- `results-new.html`

Quick command:
```bash
cd /Users/aykowolfzickler/Desktop/heartbreack-quiz
sed -i '' 's/heartbreakquiz\.com/yourdomain.com/g' index.html quiz.html results-new.html
```

### STEP 3: Deploy to Web Server (10 minutes)
Upload these files via FTP/SFTP/git:
- All `.html` files
- All `.js` files  
- All `.css` files
- `og-image.svg`
- `config.js`

**That's it! You're live.**
